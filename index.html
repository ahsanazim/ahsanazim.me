This website_

This website itself is rather curious - I decided to eschew using any third-party libraries, and stuck to plain html, css, and javascript. The javascript employed here is also a hodgepoge of random effects and functions - not exactly my usual route, as I personally swear by the excellent React.js library. Still, jquery was fun and fast to use for something of this scale, and I enjoyed the process.

If you're curious, the code is housed here on github.
